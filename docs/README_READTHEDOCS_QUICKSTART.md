# ReadTheDocs Quick Start

## 1. Connect Repository

1. Go to https://readthedocs.org/
2. Sign in with GitHub
3. Click "Import a Project"
4. Select `smrforge` repository
5. Click "Next"

## 2. Configure Project

The `.readthedocs.yml` file is already configured, so:
- **Project name**: `smrforge` (or your preferred name)
- Click "Create"

## 3. Wait for Build

- First build takes 2-5 minutes
- Subsequent builds happen automatically on git push
- Monitor build status on project dashboard

## 4. Access Documentation

Once build succeeds:
- **URL**: https://smrforge.readthedocs.io/
- Or check the URL shown in ReadTheDocs dashboard

## That's It!

The documentation will automatically update when you push to GitHub.

For detailed setup instructions, see [Read the Docs Deployment Guide](guides/readthedocs-deployment.md).

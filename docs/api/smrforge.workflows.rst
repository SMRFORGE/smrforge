smrforge.workflows package
==========================

Design of experiments, parameter sweeps, sensitivity analysis, Pareto reporting,
audit logging, surrogates, scenario-based design, and design-space atlas.

**Key exports:** ``SweepConfig``, ``SweepResult``, ``ParameterSweep``,
``full_factorial``, ``latin_hypercube``, ``sobol_space_filling``, ``random_space_filling``,
``one_at_a_time_from_sweep``, ``morris_screening``, ``sobol_indices_from_samples``,
``sobol_indices_from_sweep_results``, ``pareto_knee_point``, ``pareto_summary_report``,
``append_run``, ``RunRecord``, ``fit_surrogate``, ``surrogate_from_sweep_results``,
``run_scenario_design``, ``scenario_comparison_report``, ``build_atlas``, ``filter_atlas``.
